
# Web Scraping Caixa


Compreender a cobertatura de postos de atendimento da caixa economica: como agencias, loterias, correspondetes bancários, bem como suas particularidades a partir das bases de dados.





## Estrutura do Relatório


  * Nome Fantasia
  * Razão Social
  * Endereco
  * CNPJ
  * Agencia Vinculada
  * E-mail
  * Atividade
